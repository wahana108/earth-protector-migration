# Contributing to The Mother Earth Project (TMEP)

Terima kasih telah tertarik untuk berkontribusi pada masa depan ekonomi kebaikan! Kami sangat menghargai bantuan Anda, baik berupa perbaikan bug, penambahan fitur, maupun penyempurnaan dokumentasi.

## Cara Berkontribusi

### 1. Melaporkan Bug
Jika Anda menemukan kesalahan teknis:
- Buka "Issues" di GitHub.
- Jelaskan bug yang ditemukan secara mendetail.
- Sertakan langkah-langkah untuk mereproduksi bug tersebut.

### 2. Menyarankan Fitur Baru
Kami sangat terbuka dengan inovasi pada protokol IIP:
- Jelaskan bagaimana fitur tersebut mendukung filosofi "Give Without Loss".
- Sertakan logika matematis atau skema data jika diperlukan.

### 3. Mengirimkan Pull Request (PR)
- Fork repositori ini.
- Buat branch baru untuk fitur Anda (`git checkout -b fitur-keren-saya`).
- Pastikan kode Anda mengikuti standar yang ada di `manifesto.md`.
- Kirimkan PR dengan penjelasan singkat tentang apa yang Anda ubah.

## Standar Komunikasi
TMEP adalah komunitas yang inklusif dan saling menghargai. Mari berkomunikasi dengan sopan dan fokus pada solusi bagi Bumi.
